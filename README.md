# Nuxt3 QR Order
### 만나샵 4.0 테이블 QR주문

## Setup

yarn을 사용.

설치
```bash
yarn
```
## Development Server

Start the development server on `http://localhost:3000`:

```bash
yarn dev
```

## Production

Build the application for production:

```bash
yarn build
```

Locally preview production build:

```bash
yarn preview
```
